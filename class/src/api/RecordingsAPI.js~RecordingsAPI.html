    public copyStored(params: object): Promise<StoredRecording> source

    POST /recordings/stored/{recordingName}/copy

    Create a copy of the specified stored recording.

    API available since Asterisk 12.5

    Params:

    NameTypeAttributeDescription
    params object
    params.recordingName string

    The case-sensitive name of the recording to create a copy of.
    params.destinationRecordingName string

    The name for the new copy of the recording.

    Return:

    Promise<StoredRecording>

    Resolves with the newly copied recording when successful. Rejects if the specified recording cannot be found (status 404) or if a recording of the same name already exists on the system (status 409).

    public pause(params: object): Promise source

    POST /recordings/live/{recordingName}/pause

    Pause the specified live recording. Pausing a recording suspends silence detection, which will be restarted when the recording is unpaused. Paused time is not included in the accounting for maxDurationSeconds.

    Params:

    NameTypeAttributeDescription
    params object
    params.recordingName string

    The case-sensitive name of the live recording to act upon.

    Return:

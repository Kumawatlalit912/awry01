    NameTypeAttributeDescription
    params object
    params.username string

    The username to send with the request.

    The password to send with the request.
    params.baseUrl string

    The base url, without trailing slash, of the root Asterisk ARI endpoint. i.e. 'http://myserver.local:8088/ari'.

    public destroy(params: object): Promise source

    DELETE /mailboxes/{mailboxName}

    Destroy the specified mailbox.

    API available since Asterisk 12.1

    Params:

    NameTypeAttributeDescription
    params object
    params.mailboxName string

    The name of the mailbox to destroy.

    Return:

    Promise

    Resolves if the specified mailbox is successfully destroyed. Rejets if the specified mailbox cannot be found (status 404).

    public get(params: object): Promise<Mailbox> source

    GET /mailboxes/{mailboxName}

    Retrieve the current state of a mailbox.

    API available since Asterisk 12.1

    Params:

    NameTypeAttributeDescription
    params object
    params.mailboxName string

    The name of the mailbox to retrieve details of.

    Return:

    Promise<Mailbox>

    Resolves to the current state of the specified mailbox. Rejects if the mailbox cannot be found (status 404).

    public list(): Promise<Array<Mailbox>> source

    GET /mailboxes

    List all mailboxes.

    API available since Asterisk 12.1

    Return:

    Promise<Array<Mailbox>>

    Resolves with all mailboxes in Asterisk.

    public update(params: object): Promise source

    PUT /mailboxes/{mailboxName}

    Change the state of a mailbox. This will implicitly create the mailbox if it does not already exist.

    API available since Asterisk 12.1

    Params:

    NameTypeAttributeDescription
    params object
    params.mailboxName string

    The name of the mailbox to update.
    params.oldMessages number

    The count of old messages in the mailbox.
    params.newMessages number

    The count of new messages in the mailbox.

    Return:

    Promise

    Resolves if the operation completed successfully.

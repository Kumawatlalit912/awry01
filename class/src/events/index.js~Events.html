    public static connect(params: object): ARIWebSocket source

    Connect to Asterisk's event endpoint via websocket, and return an EventEmitter that can be used to listen for events coming from Asterisk.

    Params:

    NameTypeAttributeDescription
    params object
    params.app string | Array<string>

    The app[s] to receive events for.
    params.url string

    The ARI events url with 'ws' or 'wss' protocol, i.e. 'ws://asterisk.local:8088/ari/events'
    params.username string

    The username to use for the connection
    params.password string

    The password to use for the connection
    params.subscribeAll boolean
    • optional
    • default: true

    Whether or not to subscribe to all system events. When set to false, explicit subscription requests must be made to the ARI events endpoint to receive events for individual event sources on a given application. Param available since Asterisk 13.6
    params.reconnect boolean
    • optional
    • default: true

    Whether to reconnect to the ARI events endpoint upon unsolicited disconnect.
    params.retryOptions object
    • optional
    • default: { maxTimeout: 60000 }

    Any advanced options to pass to the 'node-retry' retry.operation() method.
    params.wsOptions object
    • optional
    • default: {}

    Any advanced options to pass directly to the 'ws' library constructor.

    Return:

    ARIWebSocket
